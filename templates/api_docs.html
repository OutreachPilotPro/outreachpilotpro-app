API Documentation

Getting Started

Our REST API allows you to integrate email discovery and outreach capabilities directly into your applications.

Base URL

https://api.outreachpilotpro.com/v1

Authentication

All API requests require an API key in the header:

Authorization: Bearer YOUR_API_KEY

Endpoints

Find Emails

POST /emails/find

{
"company": "Acme Corp",
"industry": "technology",
"location": "San Francisco"
}

Verify Email

POST /emails/verify

{
"email": "john.doe@example.com"
}

Create Campaign

POST /campaigns/create

"name": "Q1 Outreach",
"subject": "Partnership Opportunity",

}
